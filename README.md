#一.需求分析
基于Kerboros 的聊天交友软件
##1.功能模块：
①加好友

②群聊信息

③搜索功能

④聊天内容

⑤登录界面

⑥注册，修改密码+拓展（忘记密码）

##2.Kerboros
每台电脑默认已经发放固定的Kc

IDc为MAc地址

ID tgs写入固定的MAC地址

TS为获取时间函数

##3.RSA
证书信息：版本号+序列号+有效日期+主体名+公钥

用于数字签名过程

##4.数据报文
DES加解密+RAS认证

数据字典（报文格式）未确定
